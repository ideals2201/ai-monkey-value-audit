# Open-source n8n samples

The JSON files in this directory are inactive, credential-free workflows that use synthetic fixtures and make no external calls.

- `support-ticket-triage-lite.n8n.json` runs three deterministic support-ticket assertions.
- `candidate-intake-reliability-harness.n8n.json` exercises six synthetic candidate-intake reliability cases.
